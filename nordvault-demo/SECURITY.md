# Security Policy

This repository is a demonstration created for an academic bachelor thesis on Secure Software Development Lifecycle. It is not a production system and should not be deployed as one.

## Reporting a finding

If you discover a security issue in the demo code, please report it through GitHub's private security advisory feature on this repository's Security tab. Do not open a public issue.

## Supported versions

This repository is maintained for the duration of the bachelor thesis only. No long-term security support is provided after the thesis is submitted.
